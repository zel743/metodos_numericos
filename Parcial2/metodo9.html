Método regresión lineal simple y Multiple

La regresión lineal es un método estadístico para modelar la relación entre una variable dependiente y una o más variables independientes. Existen dos tipos principales: regresión lineal simple y regresión lineal múltiple. La regresión lineal simple se utiliza cuando se analiza la relación entre una variable dependiente y una única variable independiente, mientras que la regresión lineal múltiple se utiliza cuando se analizan múltiples variables independientes
